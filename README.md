## a node command template
----------
### install 
```bash
pnpm i

```
----------

### 开发

```bash
pnpm dev
```
----------

### 构建

```bash
pnpm build
```
### 单测

```bash
pnpm test
```
### 运行

```bash
// 创建全局包的链接
pnpm link --global 
// 执行包的命令
starter-node-command  -v or starter-node-command 

```
